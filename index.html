About CodeHub

CodeHub is an application which targets to make a community of coders. We want to make a code library with your help. You can add to this community by finding and adding good codes which you think might be helpful to others. You can also check what good code other people are posting and discuss about them.

Thank you Happy Coding.


Features

You can bookmark any code which you think is important.
Discuss about what other people have posted.
Upload code which you think is great directly from your mobile storage.
Create your own profile.
For more how about you check out our app.

Community of Coders

Building a community where coders share code and help other coders to understand it. Moreover by helping others you make sure that you have a strong base.

Material Design

This app was designed according to Google's Material design guidelines. Following the guidelines assured us that the end user will have a smooth, fluid and rich experience.

Community Role

A community needs a profile so get yourself one and start contributing to the community by providing good codes or by helping others understand code

Discussion

As with any community you can view other people and you can discuss with them about what they have posted in the discussion section thus clearing doubts

Time Saving

You can find your favourite codes in the bookmark section. During need just go over them so that youremember the key things which you thought are important for you.

Feedback

We value your feedback and we would love to hear about them from you. So we request you to try our app and if let us know our loopholes so that we can fix them in future updates.
